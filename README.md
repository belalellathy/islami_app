📖 Islami App
A simple and clean Islamic mobile application built with Flutter.
The app provides essential daily Islamic content such as Quran, Hadith, Tasbeeh counter, and more — with a smooth UI and structured state management.
✨ Features
📖 Read Quran (Surah list & details)
🕌 Collection of Hadith
📿 Tasbeeh Counter (Sebha)
🌙 Islamic Radio streaming
🔄 State management with Provider
💾 Local assets integration

🛠️ Tech Stack
Flutter
Dart
Provider (State Management)
Local JSON & Assets
Rest API
